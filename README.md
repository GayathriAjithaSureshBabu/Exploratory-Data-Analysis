# Exploratory-Data-Analysis
Exploratory Data Analysis (EDA) is the critical first step in data analysis where you investigate, summarize, and visualize raw data. Its primary goal is to uncover hidden patterns, spot anomalies (outliers), test hypotheses, and verify assumptions before applying any machine learning or advanced modeling.
# Retail Sales EDA

## Overview
This project performs *Exploratory Data Analysis (EDA)* on a retail sales dataset using Python. It analyzes sales trends, customer behavior, product performance, and revenue using data visualization.

## Features
- Load and clean retail sales data
- Convert date columns
- Handle missing values
- Generate summary statistics
- Visualize sales trends
- Analyze monthly sales
- Identify top-selling products

## Technologies Used
- Python 3
- Pandas
- NumPy
- Matplotlib
